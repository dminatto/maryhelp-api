# MaryHelp - API


## Getting started

- [NodeJS](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [MongoDB](https://www.mongodb.com/)

## Developing

```
git clone https://github.com/dminatto/maryhelp-api.git
```
```
cd api
```
```
npm i
```

```
npm dev
```

## Building

```
npm build
```
